# Attribute Spectrums — Placing a Brand When the Feeling Won't Come

A scaffold for the common case where a founder can't name the emotion (see SKILL → How You Converge). Instead of asking them to produce a word, you give **bipolar sliders** and have them place the brand — then you *show* what each pole looks like so they react to something concrete. Their placements triangulate a feeling even when they can't name one.

## The core spectrums

Start here; add category-specific ones as needed.

| Spectrum | Pole A | Pole B |
|---|---|---|
| Gender energy | Masculine | Feminine |
| Material world | Natural / organic | Industrial / engineered |
| Finish | Refined / precise | Sturdy / rugged |
| Temperature | Warm | Cool |
| Temperament | Playful | Serious |
| Volume | Loud / expressive | Quiet / restrained |
| Era | Vintage / heritage | Futuristic / forward |
| Density | Minimal | Maximal |
| Weight | Light / airy | Heavy / grounded |
| Register | Luxury / elite | Everyday / accessible |
| Hand | Handmade / human | Systematic / machined |

## How to run it

1. **Pick 4–6 spectrums** most relevant to the category and what you've already heard — don't run all of them.
2. **Show, don't define.** For each spectrum, put up a concrete example of *each* pole — a real image, a brand, a material, an object ("*this* is industrial: raw concrete, exposed bolts, stencil type — *this* is refined: milled aluminum, hairline rules, a quiet serif. Where are you?"). Asking a founder to define "industrial" gets you a dictionary; showing them makes them point.
3. **Have them place a dot** on each slider — and push on the *why* wherever they land hard ("all the way to industrial — what's driving that?"). The extremes and the surprises are the signal.
4. **Optionally take the stab yourself first.** Place your own dot from what you've heard and let them shove it ("I'd have put you 70% industrial — too far?"). A read to correct beats a blank slider.

## Turning placements into a feeling

The pattern across the dots *is* the emotional target in latent form. "Masculine, industrial, sturdy, cool, serious, heavy, machined" is not a random list — it's *authority through engineered force* (a feeling you can now name and design toward). "Feminine, natural, refined, warm, quiet, light, handmade" is *calm, cared-for intimacy.* Read the cluster, name the emotion it implies, then reflect it back for confirmation the normal way. The spectrums are the on-ramp to the feeling, not a substitute for naming it.

## Where it goes

Record the placements in `.agents/brand-strategy.md` alongside the muses — they justify later color/type/texture calls ("we went matte and machined because you placed hard on industrial + sturdy"). They also feed the **fixed core / flexible range**: a brand can flex along a spectrum within a range and stay on-brand.
