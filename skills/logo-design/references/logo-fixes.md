# Logo Fixes — What "Adjusting" a Mark Actually Means

For a redesign or refinement: the gap between a weak mark and a strong one is rarely "add polish." It is a specific structural move. These are the recurring ones, drawn from real before→after rebrands. Diagnose which move the mark needs before you start drawing.

## The moves

1. **Resolve ambiguity into one clear idea.** A mark that reads as *nothing* (a mirrored-letter glyph that looks like a barrel; a shape that doesn't resolve) is worse than a plain wordmark. The fix is one legible idea executed with conviction — e.g. two interlocking forms in a single circle — plus a properly spaced lockup. If you can't say in one phrase what the mark *is*, it isn't done.

2. **When the mark is weak, kill it — make the wordmark itself distinctive.** A generic abstract mark (a gradient "T", a swoosh) bolted to a plain-sans wordmark is two weak things. Dropping the mark entirely and giving the wordmark a custom letterform treatment (blocky/stencil cuts, a single altered terminal, a distinctive ligature) is frequently the stronger answer. A wordmark is not a lesser logo; a *characterful* wordmark beats a forgettable mark-plus-wordmark.

3. **Give fragile type real weight and presence.** Hairline/ultra-light wordmarks read as tentative and vanish at small sizes. Substance — a confident weight, a face with backbone — is often the whole fix.

4. **Replace a literal or cliché metaphor with a fresher, ownable one — ideally tied to positioning.** Code-braces-plus-bar-chart for a "dev stats" tool is the category's default rebus; a compass/aim mark says *direction* and ties to the brand's story. Swap the on-the-nose visual pun for a metaphor that is both less crowded and more strategic.

5. **Never ship a system default.** A wordmark set in Arial/Helvetica/Times as-drawn is not a logo, it's placeholder. Draw custom letterforms or choose a face with real character and tune it.

6. **Simplify color; drop gratuitous two-tone.** A wordmark split into two brand colors for no reason ("Dev" gray + "Stats" yellow) usually reads cleaner as one confident color, with the accent reserved for the mark or a single detail.

7. **One ownable detail can carry a wordmark.** A single sparkle on a terminal, a dropped-in glyph, a period with intent — one considered detail differentiates a clean wordmark without cluttering it. Restraint, not addition.

## The honesty rule for generated marks

Agent-drawn marks (SVG built from primitives, "parametric" constructions) tend to come out **rough** — uneven curves, awkward optical weight, near-misses that read as something unintended (a disc-with-wedge that reads as Pac-Man; a "G" that reads as "C"). Two rules:

- **A rough mark is not a resolved mark.** Say so plainly — "this is a rough first cut, the curve tension and optical balance still need work" — rather than scoring it bulletproof. Presenting an unresolved mark as finished is how the whole logo gets waved through at a quality the brand can't ship.
- **Recognition failure caps Simple** (see the SKILL body and [sad-checklist.md](sad-checklist.md)). If the mark misreads as something else at any size, that is a Simple *failure*, not a footnote — it caps the score regardless of how clean the construction looks.

## The target quality bar

A resolved mark, whatever family (wordmark / lettermark / pictorial / combination), has: **one** clear idea; a single confident construction that holds from favicon to hero; custom letterform character (not a default font); a deliberate, minimal palette; and — where a symbol earns its place — a metaphor that is distinct in the actual category and traceable to the feeling. Judge every candidate against that, in context, on the competitor shelf — never in a vacuum.
